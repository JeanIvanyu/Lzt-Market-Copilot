# LZT Market Copilot

Tampermonkey-скрипт для `lzt.market`, который оценивает стоимость лотов через `Market API`, сравнивает их с похожими объявлениями и показывает рыночные ориентиры прямо в интерфейсе маркета.

Скрипт встроен прямо в интерфейс маркета:

- На странице отдельного лота.
- В списке лотов пользователя.
- На отдельной странице настроек Copilot.

## Что Работает

- Анализ текущего лота.
- Подбор похожих объявлений через `Market API`.
- Вывод минимума рынка, средней цены, медианы и краткого вердикта.
- Вывод ближайших похожих лотов.
- Ручное обновление расчёта.
- Автоанализ на странице лота.
- Автоопределение цен в списке лотов.
- Вставка средней цены прямо в карточки лотов в списке.
- Отдельная страница настроек Copilot в боковом меню.
- Debug-режим с копированием JSON сравнения.

## Поддерживаемые Категории

Скрипт сейчас делает прямой анализ стоимости только для этих разделов:

- `Steam`
- `Minecraft`
- `Fortnite`
- `Roblox`
- `Riot Games`

Важно:

- В `Riot Games` точность пока что ниже, чем у `Steam`, `Minecraft`, `Fortnite` и `Roblox`.
- Для `Valorant` и `League of Legends` часть важных параметров нельзя надёжно восстановить из страницы и API, поэтому сравнение там пока менее точное.
- Для остальных категорий Copilot честно пишет, что прямой анализ стоимости пока не реализован, и не показывает псевдорезультат.

## Как Получить Market API Token

1. Создай API-клиент:
   [https://lolz.live/account/api/client-add](https://lolz.live/account/api/client-add)
2. Получи access token:
   [https://lolz.live/account/api/get-token](https://lolz.live/account/api/get-token)
3. При выдаче токена используй доступ `market`.
4. Открой в маркете страницу `Настройки Copilot` и вставь token в поле.

## Установка

1. Установи `Tampermonkey`.
2. Создай новый userscript.
3. Вставь содержимое файла [main.js](./main.js).
4. Сохрани скрипт.
5. Получи `Market API token` по инструкции выше.
6. Открой `Настройки Copilot` в боковом меню и сохрани token.

## Использование

### Анализ Одного Лота

Открой страницу объявления.

- Если автоанализ включён, расчёт запустится сам.
- Если автоанализ выключен, нажми `Проверить цену`.

После расчёта Copilot показывает:

- `Цена лота`
- `Минимум рынка`
- `Средняя цена`
- `Медиана`
- Краткий вывод по рынку.
- Список похожих лотов.
- Время последнего обновления.

### Анализ Списка Лотов

При переходе в раздел `Мои аккаунты` Copilot может автоматически или вручную определить средние цены для лотов в списке, включая закрытые объявления.

- Нажми `Определить цены`.
- Либо включи автоопределение в настройках.

Скрипт умеет отрабатывать и при переключении списка на закрытые лоты, даже если вкладка меняется без полной перезагрузки страницы.

## Как Это Работает

Для каждого поддерживаемого лота скрипт:

1. Считывает данные лота со страницы.
2. Определяет категорию.
3. Вытаскивает ключевые характеристики аккаунта.
4. Собирает несколько наборов фильтров для `Market API`.
5. Делает несколько запросов с разной фильтрацией.
6. Объединяет найденные объявления.
7. Ранжирует похожие лоты.
8. Считает статистику по ценам.
9. Показывает результат в интерфейсе.
